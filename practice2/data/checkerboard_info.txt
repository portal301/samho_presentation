체커보드 이미지 정보
==================

이 폴더에는 캘리브레이션용 체커보드 이미지들이 포함되어야 합니다.

필요한 체커보드 이미지:
1. checkerboard_9x6.png - 9x6 체커보드 (번호 포함)
2. checkerboard_8x6.png - 8x6 체커보드 (번호 포함)  
3. checkerboard_7x5.png - 7x5 체커보드 (번호 포함)
4. checkerboard_simple.png - 단순 체커보드 (번호 없음)

체커보드 생성 방법:
1. create_checkerboard.py 스크립트 실행
2. 온라인 체커보드 생성기 사용
3. 프린터로 출력하여 사용

체커보드 사용 팁:
- 체커보드가 평평한 표면에 놓여있어야 함
- 조명이 균일해야 함
- 카메라와 체커보드가 수직이어야 함
- 다양한 각도에서 촬영하여 정확한 캘리브레이션 수행

현재 포함된 파일:
- sample.jpg: 샘플 이미지 (체커보드가 포함된 이미지)
